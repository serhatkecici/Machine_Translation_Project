Congratulations on passing Machine Translation project 🎉

Stay Udacious!

BTW, you ask some very good questions. Please see comments below:
1 & 2) It's good to see that you're adding punctuations and special characters into your model training. We usually don't include them because these characters add dimensions which require bigger neural network and computing power. See Curse of Dimensionality here: https://en.wikipedia.org/wiki/Curse_of_dimensionality
3) You don't need to add <PAD> as another dataset. As you can see in the pad function and final_prediction function, <PAD> is just the sequence of 0.
4) This is actually handled by Keras Tokenizer function. From the documentation, this class allows to vectorize a text corpus, by turning each text into either a sequence of integers.

Models

The function simple_model builds a basic RNN model.

Suggestions:
1) reduce learning_rate from 0.01 to 0.005 or 0.001 as a smaller learning rate results in a smaller step size, which in turn improves the ability to locate the optimal point.
2) increase number of GRU units from output_sequence_length to 128 or 256.

The function embed_model builds a RNN model using word embedding.

Same suggestions from simple_model can be applied here.

Also, you can increase Embedding output_dim from 64 to 128.

The Embedding RNN is trained on the dataset. A prediction using the model on the training dataset is printed in the notebook.

The function bd_model builds a bidirectional RNN model.

Same suggestions from simple_model can be applied here.

The Bidirectional RNN is trained on the dataset. A prediction using the model on the training dataset is printed in the notebook.

The function model_final builds and trains a model that incorporates embedding, and bidirectional RNN using the dataset.

Good job for adding Embedding and Bidirectional layers in your model_final. 👍

Suggestions:
1) add encoder-decoder layers to improve your model architecture. See one-shot model from this site as an example: https://machinelearningmastery.com/encoder-decoder-models-text-summarization-keras/
2) apply the same suggestions to improve the hyperparameter settings in your model_final.
